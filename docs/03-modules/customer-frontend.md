# Customer Frontend (casino-customer-f)

## Purpose

Player-facing SPA for games, wallet, promotions, sports, and account management.

## Technology

- Angular 17 standalone components
- TypeScript 5.4
- RxJS
- i18n with XLF and locale routing
- Playwright for E2E

## Feature areas (from src/app/features)

- home
- games
- promotions
- sports
- wallet
- account
- kyc
- responsible-gambling
- ai-game-finder
- affiliate
- cms-page
- auth
- contact

## Integrations

- REST API via environment.apiUrl
- CMS API via environment.cmsApiUrl
- WebSocket for real-time updates
