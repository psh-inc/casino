# Backend - Reporting and Analytics

## Responsibilities

- Admin reporting dashboards
- Time-series analytics and daily breakdowns
- Exports (CSV, Excel, PDF)

## Key controllers

- ReportingController
- ReportingExportController
- TimeSeriesController
- DailyBreakdownController
- GameAnalyticsController, BonusAnalyticsController, ProviderAnalyticsController

## Dependencies

- PostgreSQL reporting queries
- Export libraries (Apache POI, iText, Jackson CSV)
