# Admin Frontend (casino-f)

## Purpose

Admin SPA for operations, content management, payments, reporting, and configuration.

## Technology

- Angular 17
- TypeScript 5.2
- RxJS, NGRX
- Angular Material + Tailwind
- Storybook and Playwright

## Feature modules (from src/app/modules)

- player-management
- game-management
- campaigns
- cms-admin
- cms
- reporting
- payments
- refunds
- deposit-wagering
- simple-kyc
- kafka-admin
- logs-explorer
- banners
- registration-config
- settings
- dashboard

## Integrations

- REST API via environment.apiUrl
- WebSocket for real-time updates (STOMP)
