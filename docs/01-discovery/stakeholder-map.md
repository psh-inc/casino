# Stakeholder Map

## Primary stakeholders

- Players: account, wallet, games, promotions, sports, KYC, responsible gambling
- Admin operators: player management, payments, bonuses, CMS, reporting

## Secondary stakeholders

- Compliance: KYC, self-exclusion, audit evidence
- Finance: wallet integrity, refunds, reconciliations
- Marketing: campaigns, bonuses, promotions, CRM
- Support: player issues, account recovery
- Affiliates/Partners: Cellxpert reporting, campaign attribution
- Providers: game providers, BetBy sportsbook, payment provider

## Technical stakeholders

- Platform Engineering: uptime, deployments, infra
- Security: secrets management, auth hardening
