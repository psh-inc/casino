# UI/UX User Stories

## Player (customer)

- As a player, I can register and verify email/phone so I can access the platform.
- As a player, I can deposit and withdraw funds securely.
- As a player, I can view and claim bonuses.
- As a player, I can browse games and launch sessions.
- As a player, I can set responsible gambling limits.

## Admin (operator)

- As an admin, I can search for players and view profiles.
- As an admin, I can approve or reject KYC submissions.
- As an admin, I can create and manage bonuses and campaigns.
- As an admin, I can manage game catalog and providers.
- As an admin, I can export reports in CSV/PDF/Excel.
